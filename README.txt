# WarehouseWreckage
Assets and level blueprint from WarehouseWreckage Unreal Engine project

WarehouseWreckage is a project created using Unreal Engine 5. This project was created
using Unreal's Blueprint Visual Scripting System, and therefore contains no C++ code.
Gameplay is similar to the popular video game, "Teardown", although it is heavily
simplified.

Controls: WASD for movement, Space Bar to shoot
